Phone directory CRUD API

This is an example CRUD for a phone directory API, it includes the most basic database operations via JSON.
Create a phone number

Insert a phone number into phone database.

URL:

POST http://crud.host/api/request

Parameters
Name Type Optional Description
name string True Name of the owner of the phone number
phone string True Phone number of the owner
Examples:

Example of request

POST /api/request HTTP/1.1
Content-Type: application/x-www-form-urlencoded
Accept: application/json
name=Juan Trump& phone=+52 461 111 11 11

Example of response

HTTP/1.1 200 OK
Content-Type: application/json
Status: 200 OK
{'error':false}
