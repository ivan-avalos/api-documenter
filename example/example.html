Create a phone number

Insert a phone number into phone database.

URL:

POST http://crud.host/api/request

Parameters
Name Type Optional Description
name string Yes Name of the owner of the phone number
phone string Yes Phone number of the owner
Examples:

Example of request

POST /api/request HTTP/1.1
Content-Type: application/x-www-form-urlencoded
Accept: application/json 
                                        
name=Juan Trump&
phone=+52 461 111 11 11

Example of response

HTTP/1.1 200 OK
Content-Type: application/json
Status: 200 OK 
                                        
{'error':false}
